# Comparing results for the xyz files and the postprocessing scripts

This can simply be executed by 

```ssh
python compare.py
```

The current version saves a figure and outputs a table in the terminal, it will be 100 timesteps if the GATeWAY code does not limit the total number of timesteps checked. Currently we see that some problems still exsist as not all shifts result in the same number of hydrogen bonds. Note that the origional trajectory holds the OH- in the center of the box (its shifted their for checking purposes).
